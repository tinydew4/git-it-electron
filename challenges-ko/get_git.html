Git

Git 은 리눅스 운영체제를 만든 리누스 토발즈에 의해 만들어진 무료 공개 소스 소프트웨어 (소스 코드가 공개되있습니다) 입니다.

Git 은 알려진 시간에 따른 변경사항을 추적하기 위한 프로그램이며, 프로그래밍에서 버전 제어로 알려졌습니다. 당신이 텍스트 편집 소프트웨어에서 변경사항 추적 기능을 사용하고 있다면 이미 이 개념에 익숙한 것 입니다!

Git 설치

우리는 GitHub 데스크탑 앱을 내려받아 Git 을 설치하기를 권합니다. Git-it 에서 데스크탑 앱을 사용하진 않을 것 입니다 (우리는 터미널에서 배웁니다). 그러나 그것은 Git 을 포함하고 있고 모든 플랫폼에서 같은 방법으로 Git 을 설치하는 가장 쉬운 방법입니다.

  • Windows: 터미널에서 Git 쉘을 사용하세요.
  • Mac: GitHub 데스크탑을 열고 환경설정에서 명령줄 도구 설치를 선택하세요. 터미널로 terminal 앱을 사용하세요.

이미 Git 이 있거나 확실하지 않은가요? 터미널에서 git --version 을 입력하였을 때, 보여지는 버전 숫자가 1.7.10 보다 크다면 설정된 것 입니다! 자세한 내용은 Git 웹사이트를 방문하세요.

Git 소프트웨어

GitHub 데스크탑 앱은 Git 으로 많은 것을 할 수 있지만, 전부는 아닙니다. 그래서 터미널을 배우는 것이 중요합니다. 그러나 한번 배우고나면, 데스크탑앱을 사용하는 것이 좋습니다. GitHub 웹사이트와 같이 프로젝트의 정보를 보다 시각적으로 구성해주기 때문입니다.

자체의 Git 은 컴퓨터에 있는 다른 프로그램과 같지않습니다. 데스크탑 아이콘으로 표시되지 않지만, 항상 사용가능합니다. 터미널 또는 Git 데스크탑 애플리케이션에서 언제든 접근할 수 있습니다.

Git 환경설정

GitHub 데스크탑 (과 Git) 설치가 완료된 뒤, 터미널을 여세요. 다음을 입력하여 Git 이 잘 설치되었는지 확인할 수 있습니다:

git --version

이것은 다음과 같이 Git 버전을 보여줄 것입니다:

git version 1.9.1

다음, 작업을 당신이 했음을 알게 해주는 설정을 합니다:

이름 설정하세요:

git config --global user.name "Your Name"

이제 이메일을 설정하세요:

git config --global user.email "youremail@example.com"

첫 도전을 완수했습니다! '확인' 버튼을 눌러 결과를 확인하세요.
